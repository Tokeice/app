# とけアイス

## 概要
手軽なアイスブレイクを実現した、声量判定がついているコミュニケーションアプリ。

## 操作説明
1. スマホを囲んでスタートというボタンを押して開始
2. 画面が暖色になるのを目指して、画面に表示される話題を利用しつつ会話をする
3. やめるボタンを押すと結果画面に移り、タイトル画面に戻ることが出来る
